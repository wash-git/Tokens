Adicionar Token

Esta facilidade possibilita adicionar tokens a árvore hierárquica. É realizada através de 3 etapas, utilizando-se de formulários.

Formulário 1

Possibilita ao usuário selecionar a classe PAI para a qual o token deverá pertencer. O usuário deverá clicar no botão "Selecionar".

Somente quando o nome da classe escolhida aparecer na caixa "amarela" é que a seleção da classe PAI estará concluída.

Formulário 2

Possibilita ao usuário selecionar a SUB-CLASSE, onde o token será inserido na base de dados com uma referência a esta sub-classe. O usuário deverá clicar no botão "Selecionar".

Somente quando a sub-classe escolhida aparecer na caixa "amarela" é que a seleção estará concluída.

Formulário 3

Possibilita ao usuário digitar o token e esperar a resposta do servidor informando se a operação foi bem sucedida ou houve alguma falha na operação.

A resposta do servidor será mostrada na caixa amarela.
