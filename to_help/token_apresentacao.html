Módulo Tokens da Superinterface

Bem vind@ ao Módulo Tokens da Superinterface


As facilidades aqui disponíveis ampliam o alcance da Superinterface, possibilitando interessantes análises das informações presentes nos arquivos que constituem o seu acervo. O módulo foi pensado com base no conceito de Árvore Hierárquica, que é uma estrutura de dados cujos elementos estão organizados de forma hierárquica.


Esperamos que as facilidades disponíveis neste módulo sejam de grande utilidade para seus objetivos.


Árvore Hierárquica

No nosso contexto, dados hierárquicos são uma coleção de dados onde cada item tem um único pai e zero ou mais filhos (com exceção do item raiz, que não tem pai). Toda utilização deste módulo da Superinterface se inicia na modelagem da Árvore Hierárquica que segue o modelo de conjunto aninhado ("Nested Set Model"). O analista constrói a Árvore Hierárquica de sua realidade, e o restante o módulo fará sozinho. Essa árvore é informada ao sistema através de uma planilha.


O modelo de conjunto aninhado é uma técnica para representar conjuntos de informações aninhadas (também conhecidos como árvores ou hierarquias) em bancos de dados relacionais. Cada informação ocupa um nó da árvore e estes nós são numerados de acordo com o percurso de árvore.


Integração

A Superinterface possui um corpo central que se ocupa da gestão dos arquivos do acervo, bem como módulos complementares que trazem novas funcionalidades ao sistema.


Cada módulo encapsula uma série de funcionalidades bem definidas, de forma independente, objetivando ser utilizado com outros componentes para formar um sistema mais complexo. Este módulo, ampliando as facilidades da Superinterface, gesta um conjunto de "tokens" que vem a possibilitar análises de informações contidas no acervo de arquivos da Superinterface. Também faz uso de serviços de dados prestados pelo Módulo de Verbetes.


Facilidades de Uso e Manual

Tudo foi pensado para que a utilização deste módulo fosse uma tarefa muito fácil, quase de forma intuitiva. Tuda utilização do módulo se faz através deste portal, que funciona também como um guia interativo para o usuário. De qualquer forma, estimulamos o usuário a ter conhecimento da documentação do módulo e de como construir sua Árvore Hierárquica acessando a documentação do módulo através deste documennto.
